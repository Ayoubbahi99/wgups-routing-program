# WGUPS Routing Program

This project implements a delivery routing system using:

- Hash Table (custom implementation)
- Nearest Neighbor Algorithm
- Truck simulation

## Features
- Delivers 40 packages under 140 miles
- Real-time package status lookup
- Time-based delivery tracking

## How to Run
1. Install Python
2. Run:
   python main.py
